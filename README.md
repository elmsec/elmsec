# Can Elma

building b2b ai infrastructure and licensing production-ready boilerplates. based in türkiye.

### active deployments

* **[apptork labs](https://apptork.com)**: product lab. licensing the high-concurrency backends (django 5.2) and clients (flutter/vue.js) used to scale my ventures to 600k+ users.
* **[clothshift](https://clothshift.app)**: b2b ai fashion imagery platform. core engine handles dynamic full-body image synthesis and custom model generation for 65,000+ brands.
